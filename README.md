# Internship Selection Analysis

A data analysis project focused on exploring internship selection trends, candidate performance, skills, CGPA, projects, and selection patterns using Python and data visualization techniques.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Highlights
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Candidate Selection Analysis
- CGPA & Skill-Based Insights
- Project & GitHub Score Analysis
- Data Visualization

## Key Insights
- Identified factors affecting candidate selection
- Analyzed CGPA distribution among candidates
- Compared selected vs rejected candidates
- Explored impact of projects and GitHub scores
- Visualized hiring and performance trends
